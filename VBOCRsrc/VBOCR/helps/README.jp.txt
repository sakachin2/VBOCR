V1.01 : 2017/12/14 (初版)

Windows10  Microsoft OCR Library for Windows を利用して
スキャナーで読み取ったイメージファイルから漢字かな混じり文を読み取ります
因みに "Windows Fax と スキャン" で スキャナーからイメージファイルを得ることができます

SetupVBOCR.msi を Exploler でダブルクリックしてインストールしてください
コントロールパネルでのアプリケーション名は sampleVBOCR です
MSVS2017Comunity プロジェクトソースは GitHub sakachin2/VBOCR にあります
読み取り文書の修正機能は GitHub sakachin2/VBOCR に実装してあります

ご意見、ご要望、不具合発見などは sakachin2@yahoo.co.jp まで
